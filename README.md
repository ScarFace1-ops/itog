## Описание программы

*Random Password Generator — это GUI‑приложение для генерации надёжных случайных паролей с настраиваемыми параметрами. Программа позволяет выбирать типы символов (цифры, буквы, спецсимволы), задавать длину пароля и сохранять историю сгенерированных паролей. Решает проблему создания безопасных паролей и их удобного хранения в локальном файле.*


Программа полезна:
* для быстрого создания паролей при регистрации на сайтах;
* для генерации временных паролей;
* для обучения принципам создания надёжных паролей;
* как пример реализации GUI‑приложения с сохранением данных.

## Требования для запуска

Что нужно установить на компьютер:
* Python 3.8 или выше;
* Стандартные библиотеки Python (не требуются дополнительные установки через pip):
  * `tkinter` (встроен в Python);
  * `random`;
  * `string`;
  * `json`;
  * `os`;
  * `datetime`.
